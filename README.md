# MyWebsite
該網站是練習用,使用技術: HTML CSS JS 
## Description
該專案用途為 小專題練習用
建立該網站主要是希望可以讓每位使用者,依照個人特性量身打造合適的投資建議,讓不知道如何規劃財務的朋友們有個方向

## Features
- User authentication and profile customization
- Create and share posts
- Responsive design for all devices
- 使用者身分驗證
- 個人化投資建議
- 投資風險適性度問券
- 匯率換算
